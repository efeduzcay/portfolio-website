# Personal Portfolio

A static personal portfolio website built with HTML, CSS, and vanilla JavaScript.

## Overview

This project presents professional background, selected open-source projects, technical skills, and contact information in a single-page layout. Project data is fetched dynamically from the GitHub API at runtime.

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- GitHub REST API (v3) — for dynamic project listing

## Structure

```
portfolio/
├── index.html       # Main HTML document
├── styles.css       # All styles and responsive layout
├── main.js          # Runtime logic, API calls, dark mode
└── logo.png         # Site favicon
```

## Features

- Responsive design with mobile navigation
- Dark / light mode toggle
- Dynamic project cards via GitHub API
- CV download link
